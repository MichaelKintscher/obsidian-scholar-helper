Welcome to Obsidian Scholar Helper!

Getting Started

This extension requires Obsidian and the Obsidian Scholar plugin to work. Make sure you have everything set up to import papers into your library.

  1. Install Obsidian
  2. Install Scholar community plugin (forked version)
  3. Configure your library

Now you are ready to start importing papers!

Using Obsidian Scholar Helper

Make sure you have everything set up to import papers into your library.

  1. Navigate to web page with the paper you want to import.
  2. Open the extension and click "Send paper to Obsidian".

The paper data will be added to your library, the same as if you had used the "Add paper to Library" command in the Obsidian Scholar plugin.
